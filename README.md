# ADX Market Analysis Dashboard

## Overview
This project provides an interactive dashboard for analyzing the Abu Dhabi Securities Exchange (ADX) market data. The dashboard enables users to visualize historical trends, compare company performance, and gain actionable insights.

## Features
- **Historical Data Visualization:** View market capitalization, trading volumes, and other metrics over time.
- **Company Comparisons:** Compare financial performance metrics across companies.
- **Trend Analysis:** Identify significant trends and patterns in the data.
- **Custom Queries:** Run specific queries to extract relevant insights.
- **Export Options:** Download visualizations and data for further analysis.

## Requirements
- Python 3.7+
- Pandas
- Matplotlib
- Seaborn

## Installation
1. Clone the repository:
   bash
   git clone https://github.com/your-username/adx-market-dashboard.git
   
2. Navigate to the project directory:
   bash
   cd adx-market-dashboard
   
3. Install the required packages:
   bash
   pip install -r requirements.txt
   

## Usage
1. Place the ADX dataset (`adx_data.csv`) in the project directory.
2. Run the provided Python script to generate visualizations:
   bash
   python dashboard.py
   
3. Access the visualizations and insights directly from the terminal or save them as images.

## Example Output
- Line chart showing market capitalization over time for a specific company.
- Bar graph comparing trading volumes of multiple companies.
- Summary statistics highlighting top-performing companies.
